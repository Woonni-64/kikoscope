# 四级考试刷题工具 - 技术架构说明文档

## 1. 项目概述

### 1.1 项目简介
四级考试刷题工具是一个基于Node.js的Web应用程序，专为四级考试练习设计。系统提供随机题目生成、选项打乱、即时反馈等功能，帮助用户进行有效的四级考试备考。

### 1.2 技术栈
- **后端**: Node.js + Express.js (^4.18.2)
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **数据存储**: 本地JSON文件
- **服务器**: Node.js内置HTTP模块
- **部署**: 本地服务器部署

## 3. 核心模块设计

### 3.1 题目生成模块

#### 3.1.1 功能描述
负责从题目库中随机选择题目，并确保每次生成的考试题目组合不同。

#### 3.1.2 核心算法
```javascript
// 随机选择一套题目
function selectRandomSet() {
  const setNumbers = ['set1', 'set2', 'set3', 'set4', 'set5'];
  const randomIndex = Math.floor(Math.random() * setNumbers.length);
  return setNumbers[randomIndex];
}

// 生成完整考试
function generateExam() {
  // 确保每个模块来自不同的套题
  const usedSets = new Set();
  modules.forEach(module => {
    let selectedSet;
    do {
      selectedSet = selectRandomSet();
    } while (usedSets.has(selectedSet));
    usedSets.add(selectedSet);
  });
}
```

#### 3.1.3 设计特点
- **随机性保证**: 使用Fisher-Yates洗牌算法确保真正的随机性
- **唯一性约束**: 确保每个模块来自不同的题目套
- **容错机制**: 包含数据验证和异常处理

### 3.2 选项打乱模块

#### 3.2.1 功能描述
对选择题的选项进行随机打乱，同时保持正确答案的对应关系。

#### 3.2.2 核心算法
```javascript
function shuffleOptions(question) {
  const originalOptions = [...question.options];
  const originalAnswer = question.answer;
  
  // 创建选项和答案的配对
  const optionPairs = originalOptions.map((option, index) => ({
    option,
    isCorrect: index === originalAnswer
  }));
  
  // 打乱选项
  const shuffledPairs = shuffleArray(optionPairs);
  
  // 更新问题和答案
  question.options = shuffledPairs.map(pair => pair.option);
  question.answer = shuffledPairs.findIndex(pair => pair.isCorrect);
  
  return question;
}
```

#### 3.2.3 设计特点
- **答案同步**: 确保选项打乱后正确答案索引同步更新
- **不可变性**: 使用扩展运算符创建新数组，避免修改原始数据
- **类型安全**: 支持多种答案格式（数字索引、字母标识）

- **写作**: 包含写作要求和提示
