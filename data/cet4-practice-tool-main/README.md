# 四级考试刷题工具

一个专为四级考试设计的在线刷题工具，提供随机题目生成、选项打乱、即时反馈等功能。

[![Node.js](https://img.shields.io/badge/Node.js-14.0+-green.svg)](https://nodejs.org/)
[![Express](https://img.shields.io/badge/Express-4.18.2-blue.svg)](https://expressjs.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 功能特点

### 🎯 核心功能
- **随机题目生成**: 从5套题目中随机抽取，确保每次练习都有不同的题目组合
- **选项自动打乱**: 选择题选项顺序随机打乱，避免记忆答案
- **完整题型覆盖**: 包含选词填空、段落匹配、仔细阅读、翻译、写作五大题型
- **模拟真实考试**: 按照四级考试格式设计，提供真实的考试体验

### 📝 题型说明
- **选词填空**: 1道题，考查词汇理解和上下文语境
- **段落匹配**: 1道题，考查阅读理解能力
- **仔细阅读**: 1道题，考查深度阅读理解
- **翻译**: 1道题，考查中英互译能力
- **写作**: 1道题，考查英语写作能力

### 🎨 界面特色
- **现代化设计**: 采用渐变色彩和卡片式布局
- **响应式布局**: 支持桌面端和移动端访问
- **流畅动画**: 提供良好的用户体验
- **进度显示**: 实时显示答题进度和剩余时间

## 🚀 快速开始

### 环境要求
- Node.js 14.0 或更高版本

### 安装和启动

#### 方法一：克隆仓库并启动
```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/cet4-practice-tool.git

# 进入项目目录
cd cet4-practice-tool

# 安装依赖
npm install

# 启动应用
npm start
```

#### 方法二：直接启动（推荐）
```bash
node server.js
```

#### 方法三：使用批处理文件（Windows）
双击 `start.bat` 文件

### 访问应用
启动成功后，打开浏览器访问：`http://localhost:3000`

## 使用说明

1. **开始考试**: 点击"开始考试"按钮，系统会随机生成一套题目
2. **答题**: 按照页面提示完成各个题型的答题
3. **保存进度**: 可以随时保存答题进度
4. **提交答案**: 完成答题后点击"提交答案"
5. **查看结果**: 系统会显示考试成绩和学习建议

## 故障排除

### 常见问题

1. **端口被占用**
   - 修改 `server.js` 中的 `PORT` 变量
   - 或者关闭占用端口3000的其他程序

2. **无法访问页面**
   - 确保Node.js已正确安装
   - 检查防火墙设置
   - 尝试使用 `http://127.0.0.1:3000`

3. **题目加载失败**
   - 检查网络连接
   - 刷新页面重试
   - 检查浏览器控制台错误信息
