# 英语四六级历年真题 (markdown 版)

## Why？

- 配合[「个人词库」](https://mingchang.wang/)插件学习英语。
- 网络上 PDF 及图片类资源较多，使用诸多不便。因此采用纯文本 markdown 格式，重新整理。
- markdown 是纯文本格式，尤其适合 git 管理。
- markdown 格式的排版表现力足够好。尽管不能完全复刻试卷的排版，但对学习来讲，差异可以忽略。

## 工具

主要通过**借助大模型**工具（如 Kimi、OpenAI 等），分析 **PDF 或 图片，最终转化为 md 文档**。  
当然，也有部分工作**需要人工比对**，如果可以，希望可以和大家一起**互助共享**。

喂给 Kimi 大模型的 prompt 示例如下，可根据实际情况再追问以调整到合适的程度。

```md
需求：将图片中的文字内容转化为 Markdown 格式。

- 输入：用户提供的图片文件，包含不同段落和问题的文本。
- 输出：清晰、准确、格式化的 Markdown 文本。

具体要求：

1. 内容提取：从图片中准确提取所有文本内容。
2. 格式转换：将提取的文本内容转换为 Markdown 格式。
3. 排版注意：
   - 每个段落之间需有空行分隔。
   - 非段落换行，末尾加 2 个空格。
   - 序号后的选项使用缩进格式对齐。
   - 文字字体是粗体、斜体时，输出时保证样式不变。
4. 引号使用：确保所有引号为半角格式，以适应 Markdown 语法。
5. 特殊字符：对于全角字符，需根据上下文判断是否保留或转换。如中文字符，需要保留。
6. 遇到完形填空时，不要猜测单词，直接原样展示。
7. 问题序号：保持问题序号的清晰表示，便于用户识别和阅读。
8. 文本完整性：确保文本内容的完整性和准确性，不遗漏任何细节。
```
